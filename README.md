# RL Algorithms Visualization

Interactive tool to learn reinforcement learning algorithms.

## Demo

[Live Demo](https://rl-algorithms-visualization.streamlit.app)

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Algorithms

- Q-Learning
- SARSA
- TD(0)
- n-step TD
- n-step SARSA
- Monte Carlo
- DQN (for Atari)

## Environments

- FrozenLake
- CliffWalking
- Taxi
- CartPole
- MountainCar
- Pong
- Breakout
